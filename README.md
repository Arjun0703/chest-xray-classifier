# Chest X-ray Disease Classifier

Multi-label classification of chest X-rays into 18 thoracic pathologies, built
with PyTorch and [torchxrayvision](https://github.com/mlmed/torchxrayvision).

## Models

Two architectures are supported (selectable via `--model`):

- **`densenet`** — DenseNet121 pretrained on multiple chest X-ray datasets
  (NIH, CheXpert, PadChest, MIMIC) via torchxrayvision.
- **`simple`** — a from-scratch CNN (five conv blocks, channel doubling /
  spatial halving, global pool + dropout + linear head) for studying the
  pipeline end to end.

Both output 18 pathologies — the union of labels across the source datasets.

## Conventions

- Inputs are 1-channel grayscale tensors, shape `(B, 1, 224, 224)`.
- Pixel values are in `[-1024, 1024]` (handled by torchxrayvision), not `[0, 1]`.
- Labels are floats, with `NaN` where a dataset doesn't label a given class.
- Evaluation uses ROC-AUC (`sklearn.metrics.roc_auc_score`).

## Layout

| File | Purpose |
|------|---------|
| `train.py` | Training script (data loading, training loop, evaluation) |
| `model.py` | Model definitions (`build_densenet`, `build_simple_cnn`, `build_model`) |

## Setup

```bash
python -m venv .venv
.venv\Scripts\activate        # Windows
# source .venv/bin/activate   # macOS / Linux
pip install torch torchvision torchxrayvision scikit-learn numpy
```

## Usage

```bash
python train.py --image_dir path/to/nih/images --model densenet
python train.py --image_dir path/to/nih/images --model simple
```

## Notes

Trained model weights (`*.pt`) are **not** tracked in this repository — they are
regenerated by running the training scripts.
